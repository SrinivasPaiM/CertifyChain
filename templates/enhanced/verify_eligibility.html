📋 Verification API

Or use the API endpoint directly:

POST /eligibility/verify/
Content-Type: application/json

{
  "proof": {
    "zk_proof": {
      "public_signals": [0, 50, 1]
    }
  }
}

Response:

{
  "valid": true,
  "message": "Eligibility verified - access granted",
  "what_verified": {
    "service_type": 0,
    "min_score": 50,
    "eligible": 1
  },
  "privacy_preserved": true
}
